- 일기 쓰기에 그치지 말고 내가 써내려간 하루를 읽어보며 나의 마음을 다스리고 행복한 하루를 만들고, 나아가 나 자신의 취향에 대해 알 수 있는 서비스를 만들고자 했습니다.

- 바쁜 일상 속에서도 내 친구들은 어떤 하루를 살고 있는지 공간 일기 공유를 통해 소통하는 커뮤니티 서비스를 제공합니다.
